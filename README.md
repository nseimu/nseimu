# nseimu.github.io

## Tailwind CSS

### Installation

```shell
npm install
```

### Start the Tailwind CLI build process

```shell
npm run build
```

## Static HTTP server

```shell
npx http-server ./
```

## Creating WebP Images with the Command Line

```shell
cwebp -q 50 images/flower1.jpg -o images/flower1.webp
```
